# Data Analysis of Superconductivity Dataset

This project analyzes the Superconductivity Data Set from the UCI Machine Learning Repository and builds machine-learning models to predict the critical temperature (Tc) of superconducting materials.

## Dataset
UCI Machine Learning Repository  
Superconductivity Data  
DOI: 10.24432/C5Q59B  
Link: https://archive.ics.uci.edu/dataset/464/superconductivty+data

License: CC BY 4.0  
Downloaded: 2025-12-08


## How to Reproduce

1. Download the dataset from UCI and place the `.zip` file into `data/raw/`.
2. Start with `scripts/00_prepare_data.py`.
3. Continue with scripts in order 01-05

## Software Environment

- Python 3.11.2
- kaleido 1.10.0
- pandas 2.1.1
- plotly 6.3.1
