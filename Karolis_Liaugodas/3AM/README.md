
An interactive web application for Formula 1 data analysis. Select a race session, compare driver lap times, and visualize detailed telemetry data for selected laps across different sessions.

## Requirements

You will need Python and the following libraries to run this project:
I used python 3.11.14
-   dash
-   dash-bootstrap-components
-   fastf1
-   pandas
-   numpy

You can install them using pip:
```bash
pip install dash dash-bootstrap-components fastf1 pandas numpy
```

## How to Run

1.  Execute the main application file from your terminal:
    ```bash
    python /home/karilaz/Desktop/3AM/app.py
    ```
2.  Open your web browser and go to `http://127.0.0.1:8045/`.
